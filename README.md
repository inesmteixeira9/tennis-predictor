tennis-predictor
==============================

This repository contains a tennis betting decision-making model that gives you a recommendation on betting based on the potential profit. The model uses betting odds, player rankings and surface as features to make predictions on the match outcome. The given prediction, alongside with the current betting odds, are used to provide a recomendation on betting.
